<h2>
  <a name="analysing-the-logo" href="#analysing-the-logo">
  </a>
  Analysing the logo
</h2>

<p>The logo is built up by three sticks of the same size; however, the middle one casts a shadow and is skewed.</p>

<p>Then at the bottom, it's good to note the logo is a bit rounded off.</p>

<p>My idea is to leverage borders for the two side pillars and pseudo-elements to make the skew stick + the rounded bottom section.</p>

<h2>
  <a name="recreating-the-netflix-logo-in-css" href="#recreating-the-netflix-logo-in-css">
  </a>
  Recreating the Netflix logo in CSS
</h2>

<p>Let's create a simple box and set the borders left and right to get started.<br>
</p>

<div class="highlight js-code-highlight">
<pre class="highlight css"><code><span class="nc">.netflix</span> <span class="p">{</span>
  <span class="nl">height</span><span class="p">:</span> <span class="m">15rem</span><span class="p">;</span>
  <span class="nl">width</span><span class="p">:</span> <span class="m">3rem</span><span class="p">;</span>
  <span class="nl">border-left</span><span class="p">:</span> <span class="m">3rem</span> <span class="nb">solid</span> <span class="m">#e50914</span><span class="p">;</span>
  <span class="nl">border-right</span><span class="p">:</span> <span class="m">3rem</span> <span class="nb">solid</span> <span class="m">#e50914</span><span class="p">;</span>
  <span class="nl">position</span><span class="p">:</span> <span class="nb">relative</span><span class="p">;</span>
<span class="p">}</span>
</code></pre>

<p>It's important to note the <code>3rem</code> will be the size for each stick.</p>

<p>So far, we get the following styled element.</p>

<p><a href="https://res.cloudinary.com/practicaldev/image/fetch/s--kUiPRpvP--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn.hashnode.com/res/hashnode/image/upload/v1647411594061/h7EtXSW0u.png" class="article-body-image-wrapper"><img src="https://res.cloudinary.com/practicaldev/image/fetch/s--kUiPRpvP--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn.hashnode.com/res/hashnode/image/upload/v1647411594061/h7EtXSW0u.png" alt="Two sticks styled" loading="lazy" width="442" height="570"></a></p>

<p>Let's add a before element to add the third skew line.<br>
The main idea is that this can be simply the width and height of the box.<br>
</p>

<div class="highlight js-code-highlight">
<pre class="highlight css"><code><span class="o">&amp;</span><span class="nd">:before</span> <span class="p">{</span>
  <span class="nl">display</span><span class="p">:</span> <span class="nb">block</span><span class="p">;</span>
  <span class="nl">content</span><span class="p">:</span> <span class="s2">''</span><span class="p">;</span>
  <span class="nl">width</span><span class="p">:</span> <span class="m">100%</span><span class="p">;</span>
  <span class="nl">height</span><span class="p">:</span> <span class="m">100%</span><span class="p">;</span>
  <span class="nl">background</span><span class="p">:</span> <span class="m">#e50914</span><span class="p">;</span>
  <span class="nl">transform</span><span class="p">:</span> <span class="n">skewX</span><span class="p">(</span><span class="m">21.5deg</span><span class="p">);</span>
  <span class="nl">box-shadow</span><span class="p">:</span> <span class="m">0</span> <span class="m">0</span> <span class="m">30px</span> <span class="no">black</span><span class="p">;</span>
<span class="p">}</span>
</code></pre>

<p>We can see we skew the element to fit the edges and add a bit of box-shadow to cast the shadow on the sticks below.</p>

<p>The last thing we need is the bottom rounded effect.<br>
We'll leverage a round box that we offset from the bottom.<br>
</p>

<div class="highlight js-code-highlight">
<pre class="highlight css"><code><span class="o">&amp;</span><span class="nd">:after</span> <span class="p">{</span>
